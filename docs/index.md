# 시작

## 1. 사용하기 전에

* 파스텍의 제품을 구입해 주셔서 대단히 감사합니다.
* 이 프로그램 매뉴얼에는 프로그램의 설치 방법, 사용 방법 등이 기재되어 있습니다.
* 프로그램 매뉴얼을 잘 이해하신 후에 프로그램을 안전하게 사용하여 주십시오.
* 프로그램 매뉴얼을 다 읽으신 후에는 본 제품을 사용하는 사람이 언제든지 볼 수 있도록 잘 보관해 주십시오.
